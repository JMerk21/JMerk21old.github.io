Rainbow Six Siege Pro League

  • Temporary Home Page

This site will be for information about the game, Rainbow Six Siege. It will focus mainly on the pro league,
but there will be areas for other stuff as well. The main inspiration for the site is HLTV, a site for the
professional Counter-Strike: Global Offensive scene.

The Home Page

The home page will contain a general overview of information. Similar to the HLTV site, it will have upcoming
matches to be played and articles about things going on in the community.

Upcoming Match Information

The home page will link to this page. Page will contain information about the matchup between the two teams.
It will contain info like what maps will be played, who's playing for the teams, past matchups between the two
teams.

Player Information

This page will be accessible through a link on the Upcoming Match page as well as through a search bar. The
information provided on this page will inform the user about the player they are looking at. An image of the
player will be presented. It will contain various stats about the player in their last matchups and over their time
playing professionally.

Finished Match Information

Will contain the final stats from a game. It will inform the user about the game that was played. It will contain the
scoreline of each player as well as the game's final score. Based on the score, it will also select the MVP from the
winning team and display their stats from the game next to an image on the page.

Articles

If this was a real business, there would be journalists who report on various on-goings in the community. Not just
the pro league but game information in general. I will have some pseudo-articles on the page once it gets up and
going.
