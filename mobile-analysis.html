    Introduction

    This is my mobile page review for various pages.

    Questions to consider:

    • Can you still see what you want to see?
    • Can you find what you're looking for?
    • Can you do what you want to do?
    • Any frustrations?
    • Anything work better on the small screen?

    Findings

    Can you still see what you want to see?

    For the most part the sites I used did okay. There were some that seemed too cluttered and some that just didn't work very well. There's a site I use that allows me to check my work schedule. I couldn't get it to display anything, it kept giving me errors. Amazon's mobile page, which I don't use much because I have the mobile app (which is quite nice), it seems like a shrunk down version of the desktop version... which is to be expected, I guess. I didn't like it much though, it got what it needed to get across, across. It just didn't look that nice, in my opinion.

    Can you find what you're looking for?

    Other than with the site I use to check my work schedule, I think mobile sites are getting to the point where they are working just fine.

    Can you do what you want to do?

    For the most part. I use reddit quite regularly from an unofficial android app called "Reddit is Fun". The website has a feature that allows you to hide comment chains. On the mobile app you can select the comment and it'll bring up options for the comment. You can hide, save, etc. On the mobile page you have to hit a tiny arrow to hide the comment which was a bit tedious.

    Any frustrations?

    Really only with the site I used to see my work schedule. In order to actually get the the site, I have to use the 'Request Desktop Site' button on the Chrome mobile browser. I then have to remove the "/mobile" from the url to make it work. Super annoying when I just need to see when I work.

    Anything work better on the small screen?

    Other than the issue with the comment hiding I mentioned earlier, I really like the mobile layout of reddit. It felt a lot cleaner in comparison to a desktop browser which is kind of overwhelming almost, especially if you've never used the site before.

    Summary

    I've learned to love the features that standalone apps give. I still use my mobile browser for various things, but when there's a standalone app for a particular site that I use often... I think I'll stick with the apps or the desktop versions.
